grizabella.ui (Main Components)

Grizabella UI Application.

This module contains the main entry point for launching the Grizabella PySide6 user interface.

Main function to initialize and run the Grizabella UI application. It sets up basic logging, applies the Material Design theme (light_blue.xml), creates the main window, connects signals for graceful shutdown, and starts the application event loop.

Main window for the Grizabella application. Handles the overall UI structure, including menus, views, and client interaction.
